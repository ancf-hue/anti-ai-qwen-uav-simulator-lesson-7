## 📖 О проекте

**«Оператор ШТОРМ»** — это браузерный 3D-тренажёр для обучения пилотированию беспилетных летательных аппаратов в экстремальных погодных условиях. Седьмой урок серии посвящён полёту в условиях шторма: сильный ветер, молнии, дождь и турбулентность.

Оператор должен **доставить 5 грузов** через бурю, избегая ударов молний. Количество допустимых ударов, сила ветра, частота молний и время на миссию зависят от выбранного уровня сложности.

> 💡 Проект является частью образовательной платформы **Anti-AI Qwen Academy** — серии бесплатных офлайн-приложений для развития навыков будущего.

## 🎮 Особенности

### 🎨 Три уровня сложности

| Уровень | Цвет | Время | Ветер | Молнии | Доп. ударов |
|---------|------|-------|-------|--------|-------------|
| 🟢 **Лёгкий** | `#48bb78` | 7 мин | до 10 м/с | каждые 8 сек | 5 |
| 🟡 **Средний** | `#ecc94b` | 5 мин | до 20 м/с | каждые 5 сек | 3 |
| 🔴 **Сложный** | `#e53e3e` | 4 мин | до 35 м/с | каждые 3 сек | 2 |

### 🌩️ Реалистичные погодные условия
- **Динамический ветер** — меняет направление и силу каждые 2–3 сек
- **Молнии** — с визуальными вспышками, звуками грома и уроном дрону
- **Дождь** — система частиц (300–800 капель в зависимости от сложности)
- **Турбулентность** — случайные колебания дрона в полёте
- **Облака** — объёмные 3D-облака с анимацией движения

### 🚁 Физика полёта
- Инерция и трение дрона
- Наклон корпуса при манёврах
- Ограничение максимальной скорости
- Столкновения с границами карты и землёй
- Система здоровья с визуальной шкалой

### 🎯 Система доставки грузов
- 5 зон доставки на карте
- Удержание прицела 2 секунды для сброса груза
- Визуальные эффекты захвата и доставки
- Звуковое сопровождение каждого действия

## 🎮 Управление

| Клавиша | Действие |
|---------|----------|
| `W` / `↑` / `Ц` | Движение вперёд |
| `S` / `↓` / `Ы` | Движение назад |
| `A` / `←` / `Ф` | Влево |
| `D` / `→` / `В` | Вправо |
| `Q` / `Shift` | Набор высоты |
| `E` / `Ctrl` | Снижение |
| **ЛКМ (удерживать)** | Сброс груза в зоне доставки |

## 🛠️ Технологии

- **Three.js r128** — 3D-рендеринг
- **WebGL** — аппаратное ускорение графики
- **Web Audio API** — процедурные звуки
- **Чистый HTML/CSS/JS** — без зависимостей и фреймворков
- **Адаптивный дизайн** — работает на ПК и мобильных устройствах
- **Оптимизация для слабых устройств** — автоопределение `hardwareConcurrency`

## ⚙️ Системные требования

- Современный браузер с поддержкой **WebGL** (Chrome, Firefox, Edge, Safari)
- Включённое **аппаратное ускорение**
- Рекомендуется 4+ ядра CPU для высокой производительности
- На слабых устройствах автоматически снижается качество:
  - Уменьшается количество частиц дождя
  - Отключаются тени
  - Снижается pixel ratio
